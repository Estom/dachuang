基本说明：
	TransPath工程将爬虫收集到的数据从dcspider数据库搬运到dcserver数据库，其中只搬运了只搬运了'title'、
'content'、'data_publish'、'love_count'=0、'click_count'=0、'img'字段
#######################################################################################################

Python工程介绍：
  MySQLTool.py：数据库基本操作类，集成了增、改、查询的功能
  SQLconfig.py：数据库配置文件，使用前需要先正确配置数据库基本参数
  TransPath.py：数据库搬运，从dcspider数据库中搬运数据到dcserver数据库
########################################################################################################

使用说明：
（1）、在SQLconfig.py文件中配置数据库基本参数
（2）、运行TransPath.py文件搬运数据
########################################################################################################

存在问题：
如果不改变TransPath.py代码，工程执行一次将搬运整个数据库
如果需要选择某一些数据搬运，需要配置TransPath.py的代码，配置方法见工程
